TechFlow - Responsive SaaS Landing Page

A modern, responsive SaaS landing page built entirely with HTML, CSS, and JavaScript, with no framework dependencies.

Project Features

- Fully Responsive Design - Supports desktop, tablet, and mobile devices

- Modern CSS Layout - Uses Flexbox and CSS Grid

- Interactive Features - Image carousel, parallax effect, form validation

- Performance Optimization - Optimized animations and scrolling effects

- Accessibility - Supports keyboard navigation and screen readers

- SEO Friendly - Semantic HTML and meta tags

##  How to run the project

### Method 1: Direct Access

1. Clone or download the project file.

2. Open `project/index.html` directly in your browser.

### Method 2: Local Server

```bash

# Using Python (Recommended)

cd project
python -m http.server 8000

# Using Node.js

npx http-server project -p 8000

# Using PHP

cd project
php -S localhost:8000

```

Then access `http://localhost:8000` in your browser.

### Method 3: Live Server (VS Code)

1. Install the Live Server extension in VS Code.

2. Right-click `project/index.html`.

3. Select "Open with Live Server".
